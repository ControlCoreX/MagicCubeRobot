解魔方机器人单片机端资源分配(2023年4月4日16:17:47)

控制步进电机使用 TIM3, CH1, PA6
DMA使用的是 USART2, PA2, PA3
控制舵机使用 TIM4, CH1, PD12

有一个控制步进电机使能与否的按钮, 因为使能, 所有电机可以同时使能, 同时失能, 要是所有11
个电机全部都用一个引脚控制的话, 那得很多引脚, 鉴于此, 直接用地线和vcc来控制使能失能.
而方向引脚每个电机用一个引脚控制.

步进电机的方向表示, 0表示顺时针, 1表示逆时针.
大部分电机的细分调的都是400个脉冲转一圈. 有些因为速度或其他的原因不是这样的.
单片机每秒产生800个脉冲. 如果是400个一圈的话, 则转速就是1秒转2圈.

一个函数前加了static表示这个函数只希望在本文件中使用, 不需要在外部使用. 类似文件私有的函数.

注释采用Doxygen规范:
    单行注释:   1. 该行中仅有注释:

                2. 在代码后注释:

    多行注释:   /*!

                * 2. this is comment.
                * 3. this is comment.
                */

这是一个vscode工程, 需要先搭建一下vscode开发STM32的环境.

在注释中线程跟任务不作区分.

单词的缩写标准跟上位机一样.

编译程序只需要在命令行里输入make命令并回车即可.
下载程序只需要在命令行里输入make download命令并回车即可. (下载程序时一般第一次下不进去, 这时候拔插一下ST-Link再下就能下进去了. 我也不知道为啥, 还没有去认真解决它)
